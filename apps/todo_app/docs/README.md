# todo_app ドキュメント差分

このドキュメントは `todo_app` 固有の補足・差分ルールを管理します。

## 適用方針

- 共通ルールは `docs/standards/ARCHITECTURE.md` と `docs/standards/CONVENTIONS.md` を参照します。
- `todo_app` 固有事情のみ、このファイルに追記します。

## 現在の差分

現時点では、`todo_app` 固有の追加差分はありません。

必要になった場合は、以下の形式で追記してください。

```markdown
## 差分: <タイトル>
- 背景:
- ルール:
- 適用範囲:
- 見直し時期:
```
